Inputs

BIG NOTE: Placeholders are not accessible!
The contrast on place holders is not accessible. They break the accessibility contrast rules

Buttons

  • Give button elements inner text
  • Do NOT use images for buttons, but if you do, use alt text
  • Use the button tag and not the input type="button" element,
    you will have more access to styling
  • For groups of buttons, use a fieldset to group them together.
    This helps the flow of the page, and may help those with sensory issues

    • Other

  • When a field is required, use the required tag so that screen readers will recognize it
  • Clicking the field should always do the same thing as entering the field with the keyboard
  • Clickable div and span elements are not recommended, you will lose many of the build in accessibility
    options of the form element
